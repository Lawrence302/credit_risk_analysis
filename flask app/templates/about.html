About Loan Risk Analysis App

Overview

The Loan Risk Analysis App is designed to help users evaluate the risk associated with granting loans based on various financial parameters. By inputting details such as income percentage, credit length, and loan purpose, the app utilizes machine learning algorithms to predict whether the loan application poses a high or low risk.

How It Works

  1. Users fill out a form with their financial information.
  2. The app processes the input data and generates a risk prediction.
  3. Users receive a detailed analysis, including the probability of risk and insights into their loan application.

Technologies Used

  • Flask: A micro web framework for Python, used to build the application.
  • Scikit-learn: A machine learning library for Python, used to train the predictive model.
  • Pandas: A library for data manipulation and analysis, used for handling input data.
  • HTML/CSS: For structuring and styling the web application.
